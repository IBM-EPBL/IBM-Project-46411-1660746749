This project is going to act as an interpreter deaf-mute and normal people. Nowadays, deaf and dumb people facing many difficulties in communicating with normal people. This project is an intiative to solve this problem. We have created an AI model using Convolutional Neural Network. The AI model is trained on hand gestures of American Sign Language(ASL) Alphabets. We have integrated this model with Flask UI. It can able to detect all Alphabets in ASL.
